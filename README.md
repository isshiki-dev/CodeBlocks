# CodeBlocks

Whether you need a minimalistic store for mobile users, or a full-fledged store for your SaaS, our templates have got you covered. Each template can easily be modified with our store builder.

## Features

- **AI-Powered Store Builder**: Leverage @ant-design/x to build your store through conversation.
- **On-Demand Infrastructure**: AI-automated VM workflows to boot up your store environments.
- **Optimized Performance**:
    - Streaming Markdown support (formulas, code highlighting, mermaid charts).
    - On-demand loading (tree shaking).
    - First-class TypeScript support.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **UI Components**: [Ant Design](https://ant.design/) & [Ant Design X](https://x.ant.design/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
